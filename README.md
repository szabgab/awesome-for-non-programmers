## 📊 Data Collection & Curation

> Help organize, collect, and maintain useful datasets and resource lists.

- **[List of Open Source projects](https://github.com/OSDC-Code-Maven/open-source-by-organizations/)** — A directory of open-source projects by corporations, higher education institutions, and governments. Add organizations you know about.

- **[swag-for-dev](https://github.com/swapagarwal/swag-for-dev)** — 😎👕🧦 A list of swag opportunities for developers. Submit new swag listings.

- **[Awesome Animal Care](https://github.com/omkar-foss/awesome-animal-care)** — Curated, practical resources to care for dogs, cats, fishes, horses, parrots, and others. Add care tips and resources.

- **[Awesome Fantasy](https://github.com/RichardLitt/awesome-fantasy)** — A list of fantasy books and TV series. Add your favourite reads and shows.

- **[A-to-Z-Resources-for-Students](https://github.com/dipakkr/A-to-Z-Resources-for-Students)** — A dynamic list of resources for coders and non-coders. Add and update learning resources.

- **[Useful repositories](https://github.com/Aatmaj-Zephyr/A-collection-of-useful-repositories)** — A collection of useful repositories that provide learning resources. Contribute links to helpful repos.

- **[Open Source Project and Resources](https://github.com/Ashish-khanagwal/Open-source-practice-and-resources)** — Add relevant resources that benefit your learning journey. Great for first pull requests.

- **[CheatSheets](https://github.com/Jahenr/CheatSheets)** — Cheat sheet documentation for common commands across various systems. Add or improve cheat sheets.

- **[Awesome App Ideas](https://github.com/tastejs/awesome-app-ideas)** — A list of app ideas. Non-programmers can suggest creative app ideas for programmers to build.

## 🌍 Translation & Localization

> Help make open-source projects accessible to people around the world by translating content.

- **[Cloud Native Glossary](https://glossary.cncf.io)** — Explains cloud-native concepts in simple language. Contribute by localizing/translating terms, modifying existing definitions, or proposing new terms. [Contributing Guide](https://github.com/cncf/glossary/blob/main/CONTRIBUTING.md).

- **[Godot Engine](https://godotengine.org/)** — A 2D and 3D cross-platform game engine. Help with [translating the engine and documentation](https://godotengine.org/article/godots-documentation-now-open-translation) or contribute in [other ways](http://docs.godotengine.org/en/latest/community/contributing/ways_to_contribute.html).

- **[The Turing Way](https://github.com/alan-turing-institute/the-turing-way)** — Open-source guides about data science and AI. Contribute to new/ongoing chapters, translation efforts, or infrastructure maintenance.

## 🎙️ Voice & Language

> Lend your voice or language expertise to improve tools for everyone.

- **[Common Voice](https://commonvoice.mozilla.org/en)** — Mozilla's initiative to help teach machines to understand speech. You can [contribute voice recordings](https://commonvoice.mozilla.org/en/speak), [validate recordings](https://commonvoice.mozilla.org/en/listen), and [contribute sentences](https://commonvoice.mozilla.org/en/write) in all languages.

- **[LanguageTool](https://community.languagetool.org/)** — An open-source proofreading tool. Help improve grammar and spell-checking rules. [GitHub Repository](https://github.com/languagetool-org/languagetool).

- **[Kantoniko](https://github.com/kantoniko/)** — The corner of Ladino on the Internet. Contribute voice recordings, words, translations, and sentences in Ladino.

## ✍️ Content Creation & Writing

> Write, document, share recipes, jokes, or knowledge — no coding required.

- **[DevJoke](https://github.com/shrutikapoor08/devjoke)** — #DevJoke. Submit a PR if you know a good dev joke!

- **[the-sourdough-framework](https://github.com/hendricius/the-sourdough-framework)** — Share your bread and sourdough recipes with the community.

- **[Open Drinks](https://github.com/alfg/opendrinks)** — Submit drink recipes for an open-source recipe collection.

- **[Data Science Notes](https://github.com/wyattowalsh/data-science-notes)** — Share notes across data science topics such as mathematics, visualization, and modeling.

- **[PetMe](https://github.com/akshitagupta15june/PetMe)** — An awesome pet-care repository. Contribute blog posts and content during Hacktoberfest and beyond.

- **[Basic GitHub commands](https://github.com/LiviaGitHub/basic-github-commands)** — A comprehensive guide to fundamental Git commands. Add new chapters, expand existing ones, or work on translations.

- **[DX Knowledge Base](https://github.com/DXHeroes/knowledge-base-content)** — Open-source knowledge base about Developer Experience. Contribute articles about software creation processes and team management.

- **[LaTeX Gboard Dictionary](https://github.com/DenverCoder1/LaTeX-Gboard-Dictionary/issues/5)** — ⌨ Add shortcuts to a Gboard dictionary to allow Unicode to be typed with simple shortcuts on Android.

## 📋 Contributing to This List

Want to add a project? Please follow these guidelines:

1. **Fork** this repository and create a new branch.
2. **Add your project** under the appropriate category with this format:
   ```
   - **[Project Name](URL)** — Brief description of what non-programmers can do. [Contributing Guide](URL).
   ```
3. Make sure the project is **actively maintained** and has a clear way for non-programmers to contribute.
4. **Submit a pull request** with a brief explanation of why this project belongs here.

> **Tip:** If you're not sure which category fits, just pick the closest one — we can always adjust during review!
